# SAVE ME
> 유기동물 입양 & 실종동물 찾기 플랫폼
## 요약
     포인핸드를 벤치마킹
     
     * 반료동물이슈 및 통계 조회
     포인핸드 블로그 내 '반려동물이슈'정보를 조회할 수 있습니다
     오늘 구조된 유기동물의 수와 안락사율, 입양률을 한번에 확인할 수 있습니다.
     
     * 전국보호소  유기동물 정보 실시간 조회
     사용자는 실시간으로 전국 보호소 유기동물 정보를 조회할 수 있습니다.
     
     * 실종/보호/목격 신고기능
     반려동물을 잃어버렸을 경우 사진과 함께 글을 등록할 수 있습니다. (실종)
     유기동물을 발견했을 경우 사진과 함게 글을 올려 보호자가 찾을 수 있도록 도와줄 수 있습니다. (보호/목격)
     실종동물이 주인을 찾았을 경우 상태를 완료로 변경할 수 있습니다.
     
     * 커뮤니티
     입양후기, 입양홍보, 임시보호 등 카테고리 별로 글을 작성하여 소통할 수 있습니다.
  

## 개발환경
> 프론트엔드
* IDE : AndroidStudio
* 언어 : Kotlin
* HTTP 통신 : Retrofit2
> 백엔드
* IDE : Pycharm
* 언어 : Python
* 프레임워크 : Django
* 데이터베이스 : MySQL
